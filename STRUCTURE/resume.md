项目名称：

ReflexCode —— 基于 LangGraph 与 MCP 的自反思代码生成 Agent

项目描述（4 条左右）：

基于 LangGraph 构建 Code → Review → Retry 状态机，利用条件边实现自动修复循环，并通过 Retry Budget 防止无限迭代。
基于 MCP（Model Context Protocol） 实现 File 与 Git 工具服务，设计统一 Tool Schema、权限白名单及操作审计机制，保障 Agent 工具调用安全性。
集成 DeepSeek SDK 与流式输出能力，实现代码生成、Review 和工具调用的统一 Agent Runtime，并支持执行状态实时推送。
构建评测脚本，对典型代码生成任务统计成功率、平均重试次数、Token 消耗及执行耗时，为 Agent 效果优化提供数据支撑。

这四条已经足够覆盖大部分 Agent 岗位 JD。
